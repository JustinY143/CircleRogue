#CircleRogue
A roguelite bullet heaven where you shoot dots...

How to Play
Controls:
WASD to move
Mouse to aim
Left Click to attack
ESC to pause

#Classes
Gunner: Ranged attacks with bullets
Swordsman: Melee attacks with area slashes

#Play Online
The game is hosted on GitHub Pages. Just open the link and play in your browser.

#Development
To run locally:
Clone the repo
Open index.html in a browser

Files:
index.html - Main game page
game_state.js - Game progression and saves
entities.js - Player, bullets, particles
enemies.js - Enemy types and AI
game_loop.js - Main game update/draw loop
ui.js - Menus and HUD
utils.js - Settings and helper functions
